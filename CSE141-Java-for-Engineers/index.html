Teaching Approach

Visualization over explanation

Abstract concepts taught with live tools — not just slides

Python Tutor · pythontutor.com
2D Array Memory Layout
Used live in class to show students how int[][] X maps to memory — rows, column indices (0,0), (0,1)... and the pointer from the variable to the array object. Students could paste their own code and step through execution.
JShell · Java Interactive REPL
Array Default Initialization
Demonstrated interactively that int[] x starts as null before allocation, and becomes {0,0,0,0,0} after new int[5]. Students ran it themselves in real time — no waiting to compile a full class.

Why the examples look like circuits

These students take power electronics and circuit analysis in the same year they take this programming course. Using generic examples — bank accounts, to-do lists — would have worked, but would have missed the opportunity to show them that the tools they're learning connect directly to the engineering they already care about.

The RLC simulation uses the same numerical method (Forward Euler) that MATLAB and EMTP use for power systems transient analysis. Students implement it in 40 lines of Java.

Math.sqrt, atan2 → Impedance magnitude & phase
double[], loops → RC discharge v(t) = V₀e^(-t/RC)
2D arrays → Nodal analysis matrix (Cramer's rule)
Euler integration → RLC state-space simulation
ArrayList + Stack → Exam score processor (real grading)
