## Objective
Build and verify the behavior of an SR latch using cross-coupled NOR gates.

## Theory
The SR latch is the simplest sequential element — it stores one bit.

| S | R | Q | Description |
|---|---|---|-------------|
| 0 | 0 | Q | Hold (memory) |
| 1 | 0 | 1 | Set |
| 0 | 1 | 0 | Reset |
| 1 | 1 | — | Forbidden |
