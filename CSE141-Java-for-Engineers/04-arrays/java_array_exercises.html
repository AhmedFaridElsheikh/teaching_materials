Java Array Exercises

Easy 1 — Find the maximum element
▼
Task

You have an array of integers. Find and print the largest number in the array.

Example: {3, 7, 1, 9, 4} → prints 9

Starter code
public class Exercise1 {
    public static void main(String[] args) {

        int[] arr = {3, 7, 1, 9, 4};

        // Write your code here
        // Hint: use a variable to track the max

    }
}
Hint

Create a variable int max = arr[0]; before the loop. Inside the loop, if arr[i] > max, do max = arr[i];. After the loop, print max.

Solution
public class Exercise1 {
    public static void main(String[] args) {

        int[] arr = {3, 7, 1, 9, 4};

        int max = arr[0];
        for (int i = 1; i < arr.length; i++) {
            if (arr[i] > max) {
                max = arr[i];
            }
        }

        System.out.println("Max: " + max);
    }
}
Easy 2 — Count even numbers
▼
Task

You have an array of integers. Count how many of them are even and print the result.

Example: {1, 2, 3, 4, 5, 6} → prints 3

Starter code
public class Exercise2 {
    public static void main(String[] args) {

        int[] arr = {1, 2, 3, 4, 5, 6};

        // Write your code here
        // Hint: a number is even if number % 2 == 0

    }
}
Hint

Start with int count = 0; before the loop. Inside the loop, check if (arr[i] % 2 == 0) — if true, do count++;. Print count after the loop.

Solution
public class Exercise2 {
    public static void main(String[] args) {

        int[] arr = {1, 2, 3, 4, 5, 6};

        int count = 0;
        for (int i = 0; i < arr.length; i++) {
            if (arr[i] % 2 == 0) {
                count++;
            }
        }

        System.out.println("Even count: " + count);
    }
}
Medium 3 — Print array in reverse
▼
Task

You have an array of integers. Print all its elements in reverse order, each on a new line.

Example: {1, 2, 3, 4, 5} → prints 5 4 3 2 1 (each on its own line)

Starter code
public class Exercise3 {
    public static void main(String[] args) {

        int[] arr = {1, 2, 3, 4, 5};

        // Write your code here
        // Hint: think about where your loop should start and end

    }
}
Hint

Instead of starting your loop at i = 0, start at i = arr.length - 1 and go down: i-- instead of i++. Stop when i >= 0.

Solution
public class Exercise3 {
    public static void main(String[] args) {

        int[] arr = {1, 2, 3, 4, 5};

        for (int i = arr.length - 1; i >= 0; i--) {
            System.out.println(arr[i]);
        }

    }
}
